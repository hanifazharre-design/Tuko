## Menjalankan Aplikasi Secara Lokal
Aplikasi ini dibagi menjadi 2 *folder* utama:
1. `Front-End` (React + Vite) - Gunakan perintah `npm run dev` untuk menjalankan tampilan antarmuka.
2. `Back-End` (Node.js + Express + Prisma) - Gunakan perintah `npm run dev` untuk menjalankan sisi peladen (API) dan database.
